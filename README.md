### Integrated Android Ecosystem for Student Wellness & Sustainability

##  Project Overview
University students often neglect their well-being due to academic pressure. DataSelfie is a native Android application designed to solve this by centralizing health tracking (sleep, hydration) and academic organization in a single interface. It bridges the gap between productivity and physical health through interactive tools and gamification.

##  Key Features
* Wellness Tracker: Monitors daily sleep cycles and hydration levels to prevent burnout.
* Academic Organizer: Manages exam schedules and pending tasks efficiently.
* Gamification: Built-in Memory Game developed with custom logic to improve cognitive skills and user retention.
* Sustainability Module: Encourages and tracks eco-friendly daily habits.

##  Tech Stack
* Language: Java (Core Logic & OOP Principles).
* Interface: Android XML Layouts.
* IDE: Android Studio.
* Version Control: Git & GitHub.

##  Technical Implementation
* Implemented Object-Oriented Programming principles to manage the complex state of the Memory Game.
* Designed custom XML layouts to ensure a responsive and user-friendly experience.
* Structured the project following standard Android development practices for scalability.
